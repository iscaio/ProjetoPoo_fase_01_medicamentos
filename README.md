# Projeto de Estudo: Gestão de Medicamentos Hospitalar e QuickSort

Exercicio da faculdade desenvolvido para a disciplina de POO avançado. O estudo é foca na aplicação de conceitos de Programação Orientada a Objetos (POO) e algoritmos de ordenação em Java.

## Funcionalidades

O sistema cumpre os seguintes requisitos:

1. **Adicionar:** Registro de medicamentos na lista.
2. **Remover:** Exclusão de itens por índice.
3. **Recuperar:** Busca de item por atributo (Código).
4. **Listar:** Exibição de todos os itens criados.
5. **Contar:** Retorno do total de itens na base.
6. **Ordenar:** Organização por Dosagem Mg via algoritmo QuickSort

## Ferramentas e Conceitos

- **Linguagem:** Java.
- **POO:** Herança, Abstração, Polimorfismo e Interfaces.
- **Estrutura de Dados:** Manipulação de ArrayList e conversão para Arrays estáticos.
- **Algoritmos:** Implementação manual do QuickSort (Lomuto).
